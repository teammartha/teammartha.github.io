What is pokeheartgold-expanded?

This project is a decomp hack of Pokemon HeartGold and Pokemon SoulSilver!
Because the pokeheartgold project is still fairly work in progress, this project may also develop into a hybrid decomp/ROM hack.

How will this project be distributed?

This project currently has no releases, and the distribution method will change depending upon features added.
The final project will likely be distributed as an IPS patch given the gray area of hosting ROM hacks on GitHub.
If tools to develop the whole project in decompilation eventually arise, this may change.

What broad features does it have?

The design goals of this hack are:
-- Pokemon new to Gold and Silver (Larvitar, Sneasel, etc.) can be caught before Lance, and Hoenn/Sinnoh Evolutions are now all usable before Lance
-- Level scaling for routes and major battles improved, with more choices in the early game
-- Select Kanto/Johto Pokemon receive buffs, making unpowered Pokemon usable in ways mostly consistent with Gamefreak's style of buffing
-- Helpful resources for teambuilding available more easily (low cost battle items, etc.)
-- All 493 Kanto, Johto, Hoenn and Sinnoh Pokemon catchable in one game
